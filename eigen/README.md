# Eigen3 Examples and Tutorials

Basic PowerShell compilation:

`PS> cmd.exe --% /c g++ -Wall -Wextra -I "%LOCALAPPDATA%\Programs\eigen-cpp" -g example2.cxx -o bin\example2`


Run from PowerShell:

`PS> .\bin\example.exe`
